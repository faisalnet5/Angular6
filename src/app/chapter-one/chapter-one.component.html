  • Lesson Two: Databinding - Communication between TS code (Business Logic) and HTML (Template)
    • Output Data: From TS code to HTML
      • String Interpolation: {{ data }}
        Example-1: {{message_one}}
        Example-2: {{getMessage()}}

      • Property Binding: [property]="data"
        Note: [...] indicates to Angular that we are using property binding that we want to dynamically bind a property of current element.

        Example: (This button will be active in 5 second due to property buinding)

    • React to user events: From HTML to TS code
      • Event Binding: (event) = "expression"
        Note: (...), through this we are trigering an event (can be button click, keyUp, etc.,) in Angular.

        Example-1:      >> {{show_message}}

        Example-2:  (on blur, pass information)    {{enteredMessage}}

    • React to user events and output data at the same time
      • Two-Way-Binding: [(ngModel)] = "data"
        Example:    Your entered data will appear while typing > {{message_three}}
